WS宏 是一款魔兽世界PVP辅助软件，目前仅对外开放狂暴战版本，当前版本支持魔兽世界9.27
目前支持自动最优循环输出，防骗断(目标不施法时或无法打断时无法施放打断技能)，自动重要引导技能秒断
WS宏是基于WA的图像识别，按键方式比目前同类软件都更为安全，无法被官方检测
目前测试2-3个月无任何形式的魔兽世界官方惩罚行为，可放心使用

使用说明
1.要求魔兽世界及Windows操作系统为简体中文版本，暂不支持其他语言，包括英语、繁体中文等
2.要求魔兽世界以全屏的方式运行(屏幕不允许有游戏标题框，游戏填充整个显示器)
3.要求魔兽世界运行于主显示器(如有双显示器需运行于第一屏幕)
4.要求游戏内安装WA(WeakAura)魔兽世界插件
5.游戏内导入WA字符串，字符串在网页下方
6.运行WS宏程序，程序名为 "WowSc_版本号.exe"，如提示需要下载微软.Net运行框架，按照提示下载后重新运行即可
7.使用全局按键 Ctrl + F12 可以关闭后台运行的 WowSc

游戏内设置
1.游戏内动作条按键 F1 设置为 拳击
2.游戏内动作条按键 F2 设置为 竞技场目标1拳击 (新建魔兽世界宏 /cast [@arena1]拳击)
3.游戏内动作条按键 F3 设置为 竞技场目标2拳击 (新建魔兽世界宏 /cast [@arena2]拳击)
4.游戏内动作条按键 F4 设置为 竞技场目标3拳击 (新建魔兽世界宏 /cast [@arena3]拳击)
5.游戏内动作条按键 F5 设置为 焦点拳击 (新建魔兽世界宏 /cast [@focus]拳击)
6.游戏内动作条按键 F6 设置为 暴怒
7.游戏内动作条按键 F7 设置为 斩杀
8.游戏内动作条按键 F8 设置为 怒击
9.游戏内动作条按键 F9 设置为 嗜血
10.游戏内动作条按键 F10 设置为 旋风斩
11.游戏内动作条按键 F11 设置为 乘胜追击
12.WS宏当前默认 F 键位为手动当前拳击，用户动作条 F键位 不应该放置任何技能
13.WS宏当前默认 Alt+A 为手动竞技场目标1拳击，用户动作条 Alt+A 键位不应该放置任何技能
14.WS宏当前默认 Alt+S 为手动竞技场目标2拳击，用户动作条 Alt+S 键位不应该放置任何技能
15.WS宏当前默认 Alt+D 为手动竞技场目标3拳击，用户动作条 Alt+D 键位不应该放置任何技能
16.选择目标后，按1键会自动开启最优攻击循环，使用其他键位（打断键位，移动，跳跃等键位除外）会停止自动攻击而施放其他技能
